# Auto_Repo_Downloader
Automatically download or update git repos. Only tested on Ubuntu 16.04.

## Usage
1. Satisfy requirements by running ```requirements.sh``` first.
2. Paste git URLs into ```source.txt```. One URL per line.
3. Run ```download.py```.

## Supported URLs
* GitHub repos
* .tar.gz files
* .tar.bz2 files
* SVN repos
* SourceForge repos
* .zip files
